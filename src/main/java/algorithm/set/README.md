# Set
## Tree Set
- TreeSet 중복 허용 x, 정렬 (red-black-tree 활용)
  - new TreeSet<>(Collections.reverseOrder());
    - 내림차순 정렬
    - remove(value) 원소 제거
    - size() set의 크기
    - first() 정렬된 첫번째 원소값 반환
    - last() 정렬된 마지막 원소값 반환
