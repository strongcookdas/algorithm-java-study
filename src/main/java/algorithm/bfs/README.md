# BFS

## 0-1 BFS

```angular2html
가중치가 0과 1 밖에 없는 그래프에서 최단 경로를 찾기 위한 알고리즘
덱을 사용한다.
```

## BFS 문제 풀 때 주의사항

- 큐 크기만큼 반복문 순회시 Q의 크기는 변수에 저장하여 반복문을 순회
    - for(int i = 0; i<Q.size(); i++) 이렇게 하면 안된다.
- 방문 체크 q.contains로 하면 안됌 (poll하기 때문)
