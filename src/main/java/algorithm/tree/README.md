# Tree

## 관련 지식

### 트리 구현 방법

- 재귀 함수를 통해 부모 노드에 자식 노드 추가
    - 장점
        - 트리의 크기가 동적인 경우 편리
        - 트리의 구조를 직관적으로 알 수 있음
    - 단점
        - 포인터 저장에 의한 메모리 낭비
        - 재귀 호출 오버헤드
- 배열을 이용한 트리 구현
    - 장점
        - 연속된 메모리를 사용함으로써 캐시 효율성 증가
        - 인덱스를 이용하여 빠르게 노드 접근
    - 단점
        - 동적 확장 어려움
        - 노드 삭제 복잡
