# Hash
##  HashMap 함수
- size() map의 길이
- containsKey('key') key 확인
- remove('key') 키 제거
- keySet() 키 집합
- getOrDefault(key, defaultValue) 키에 해당하는 값 반환 혹은 기본값 반환
- equals(map) map 끼리의 비교
