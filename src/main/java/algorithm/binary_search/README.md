# 이진 탐색

## 주의 사항

- 반복문, 조건문의 조건이 복잡하면 반례 케이스에 걸릴 확률 높아짐
- 단순하게 코드를 구현하도록 연습이 필요
- mid 값과 answer 값을 분리
    - answer를 mid로 사용하는 경우 답이 이상하게 나올 확률 높아짐
- 타입 경계값 확인 필요
- int * int 시 타입변환 필요없는가 확인
- rt 값 초기화 주의
